What is the FMS File Upload App

The FMS File Upload App is the graphical user interface application for uploading files into AICS's centralized file storage colloquially known as FMS. To search for or access the uploaded files now in FMS you can use another AICS application, the FMS File Explorer App.

Files in FMS are best located by their 'annotations' (metadata about the file ex. I can upload an imaging file and annotate it with the annotation "Program" giving the value that corresponds with the project I'm working on e.g. EMT, Nucmorph, etc.). Every file in FMS is, by default, given some basic annotations ("Uploaded by", "Kind", "Type"), but to better differentiate your files it is helpful to add additional annnotations which the FMS File Upload App allows you to do.

    How to use while working from home

    • To use outside of the office, connect to the Allen Institute VPN.
    • N.b.: This application requires access to the public internet to work properly. If your VPN configuration blocks access to the public internet, you'll notice--among other potential issues--that usages of certain icons are replaced by placeholder boxes. Those boxes are still functional, but without seeing the icon, it may be difficult to know what function the boxes serve!

    Tutorials

      How to find uploaded files

      Check out the FMS File Explorer App, it is another AICS application in which you can search for files by basic information, like their name or who uploaded it, as well as any other metadata the file has been annotated with.

      Note: Metadata about uploaded files can also be seen in the File Upload App by the user whom uploaded them. The metadata can also be updated in this way.

      How to report an issue or otherwise get in touch

      If you'd like to bring up something that needs immediate attention i.e. is blocking your work, chat with us on the #support_aics_software channel on Slack.

      Otherwise, submit a ticket using our JIRA request form! Must be on-site or connected to the institute via VPN to access JIRA. Likely by the next week you can expect someone from our team to reach out to you or someone listed in the ticket to gather more information, but feel free to reach out to us, the software engineering team, before then (e.g., Slack, email, Zoom, Teams, etc).
